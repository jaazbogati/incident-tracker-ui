# Incident Tracker UI

A modern frontend dashboard for managing incidents, users, and analytics, built with React and Tailwind CSS.

---

## 🚀 Live Demo

Frontend: https://incident-tracker-ui.onrender.com
Backend API: https://incident-tracker-api-f0cj.onrender.com

---


## 🧩 Features

* 🔐 Authentication (JWT-based)
* 📊 Dashboard with metrics & charts
* 📁 Incident management (CRUD)
* 👥 User management (Create, Edit, Delete)
* 📈 Reports & analytics
* 🔎 Search & filtering
* 🎨 Responsive UI with Tailwind CSS

---

## 🛠 Tech Stack

* React (Vite)
* Tailwind CSS v4
* Axios
* React Router
* Recharts
